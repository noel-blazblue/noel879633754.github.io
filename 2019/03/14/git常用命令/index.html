git常用命令

Posted on 2019-03-14

1、查看用户名和地址
git config user.name
git config user.email

2、修改用户名和地址
git config –global user.name “your name”
git config –global user.email “your email”

创建一个本地仓库
git init

添加文件到缓存区
git add

把缓存区文件添加到仓库
git commit -m “message”

查看当前修改状态
git status

查看修改后的不同点
git diff

查看版本提交日志
git log
git log –pretty=oneline

回退到某个版本
git reset –hard HEAD^ //上个版本 ^^为上上个版本
git reset

丢弃暂存区的修改
git reset HEAD

撤销修改
git checkout –

记录每一次命令
git reflog

删除文件
git rm

关联远程github仓库
git remote add origin git@github.com:/.git

查看远程仓库
git remote -v

推代码到远程仓库并关联分支
git push -u origin master

推代码到远程仓库，无需关联
git push origin master

克隆一个远程仓库到本地
git clone git@github.com:/.git

创建并切换到一个分支
git checkout -b
||
git branch 创建分支
git checkout 切换分支

合并分支
git merge

删除分支
git branch -d

查看分支合并图
git log –graph

合并分支并提交一个新的commit
git merge –no-ff -m “merge with no-ff” dev

保存现有工作区
git stash

查看保存的工作区
git stash list

恢复工作区
git stash pop

恢复指定工作区
git stash apply stash@{0}

查看远程仓库
git remote -v

推送分支到远程仓库
git push origin

指定本地与远程分支的链接
git branch –set-upstream-to=origin/dev dev

建立本地和远程分支的关联
git branch –set-upstream branch-name origin/branch-name

查看标签
git tag
添加标签
git tag
添加指定标签
git tag
删除标签
git tag -d

推送某个标签到远程
git push origin

删除已有远程库
git remote rm origin

推送到指定远程库
git push github master
